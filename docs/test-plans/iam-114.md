# Test Plan: IAM-114 Board Layout Improvements

## Scope

Verify the user-facing board layout improvements described for IAM-114 on the live Kanban Board app:

- Board title feels visually heavier and more prominent than the nearby search and filter controls.
- Each column header shows a colored bottom accent line that matches the column color.
- Each column header shows a small colored ticket-count badge.
- Empty columns show a placeholder with a square icon and `No tickets` text.
- Ticket cards still open the ticket detail modal.
- The warm cream board palette remains unchanged.

Out of scope:

- Source-code or DOM-implementation inspection.
- Backend API correctness beyond what is directly visible in the UI.
- Mobile and tablet layout behavior.

## Test Approach

- Black-box Playwright execution against `http://localhost:5173`, then `http://127.0.0.1:5173` if needed.
- Visual inspection using browser snapshots and screenshots.
- No source-code inspection.

## Test Data Strategy

- Use existing board data only.
- Do not create, edit, or delete tickets.
- If an empty column already exists, use it to verify the empty state.
